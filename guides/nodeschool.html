NodeSchool

NodeSchool is an open source curriculum that teaches Node.js through adventures that run in your terminal. While these adventures are self-guided, don't hesitate to seek help from a wizard if at all necessary.

Also, this guide is a work in progress, if you find a problem with it, please make an issue .

Getting started

To get started, you first need to install Node.js. Once you have Node.js, use npm (it comes with Node.js) to install the adventure you want to start. Once the adventure is installed, simply type the adventure's name to launch.

You will also need a Text Editor. If you don't have one already, you may want one for editing code. A few options: Atom, Sublime Text, Light Table, Textmate, Brackets.

Core Adventures

These adventures focus on base Node.js magic spells.

Stuck? Find a local wizard or ask a question in the remote discussion.

javascripting

Start your JavaScript craft. No previous programming experience required.

npm install -g javascripting

learnyounode

Learn some introductory Node.js magic: asynchronous i/o, http.

npm install -g learnyounode

stream-adventure

Adventure into the cyberwizardry of composable streams.

npm install -g stream-adventure
cd ~/nodeschool

Elective Adventures

These adventures target popular spells or styles of magic.

Stuck? Find a local wizard or ask a question in the remote discussion.

browserify-adventure

Use the magic of npm modules and node-style require() in the browser with browserify.

npm install -g browserify-adventure
browserify-adventure

For a more advanced overview of browserify, check out the browserify handbook.
